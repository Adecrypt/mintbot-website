🚀 Getting Started

Mint_bot runs entirely inside Telegram. No app download, no browser extension needed.

1
Open the botSearch @ade_mintbot on Telegram or tap the button above.
2
SubscribeTap 💳 Subscribe from the menu. $5 trial (7 days) or $15/month.
3
Add a walletTap 💼 Wallet → generate or import. Send ETH for mint price + gas.
4
Track & mintPaste an OpenSea URL or contract address. Set up Auto-Mint and you're live.

💼 Add a Wallet

Keys are encrypted in storage. Only add funds you're ready to deploy.

Tap 💼 Wallet from main menuGenerate a new wallet or import an existing one with your private key.
2
Fund itSend ETH to your wallet address. Minimum ~0.01 ETH recommended for gas.
3
Add more for multi-walletAdd up to 15 wallets and fire them all simultaneously on a drop.

🤖 Auto-Mint

Tap 🤖 Auto-Mint on a tracked project → choose gas → choose quantity. Bot watches 24/7 and fires when mint opens.

🐢 Low → 90% of base fee
⚡ Normal → 120% of base fee
🚀 Fast → 150% of base fee
🔥 Rapid → 200% (next block guaranteed)

Use Fast or Rapid on competitive mints. Low is fine for free mints with no competition.

⚡ Mint Now

When mint is live, an ⚡ Mint Now! button appears on the tracked project. Tap → gas → quantity. Fires instantly with pre-built transaction. No setup needed.

👛 Multi-Wallet

Add and fund multiple wallets, then tap 👛 Multi-Wallet on any tracked project. All wallets fire simultaneously the moment a Transfer is detected.
